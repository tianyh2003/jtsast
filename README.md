# jtsast

仓库地址：[tyh4567/jtsast (github.com)](https://github.com/tyh4567/jtsast/)

## 推荐使用环境

ubuntu 22.04  
node 18.20.2

## 快速开始

### 安装

```
git clone https://github.com/tyh4567/jtsast
cd jtsast
npm install  
npm link
```

### 验证

```
jtsast
```

出现以下则成功

```
# jtsast
# input file address : ./in/
# output file address : ./out/
```
